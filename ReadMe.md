# MyWebApp 🚀

Fullstack application built with ASP.NET Core Web API (backend) and Flutter (frontend).  
The project demonstrates communication between mobile app and server via REST API.

---

## 🧩 Tech Stack

**Backend:**
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- DTO pattern
- Service layer architecture
- CQRS / Factory patterns

**Frontend:**
- Flutter (Dart)
- HTTP package for API communication

---

## 📌 Features

- User management (CRUD)
- Patient management (CRUD)
- REST API backend
- Flutter mobile app consuming API
- DTO-based data transfer
- Layered architecture (Controllers → Services → Data)

---

## 🔗 Architecture

Flutter App → HTTP Requests → ASP.NET Core API → SQL Server
